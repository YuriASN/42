file .h Â e' a lista (lib) das suas funcoes.
	Dentro da lib eu coloco a primeira linha da(s) funcao(oes).
	isso faz com que a compilacao confira se os argumentos enviados sao compativeis com argumentos que a funcao recebe.
A declaracao de uma lib deve corresponder a funcao .c existente.

Para proteger as libs de loops (deve-se sempre fazer), e' necessario criar um IFNDEF com
	o nome da lib modificado (__FT_nomedalib_H__) e dentro dele definir o mesmo ifndef.
	assim depois de entrar no indef uma vez ele nao voltara a mesma lib pois o IFNDEF foi definido.
